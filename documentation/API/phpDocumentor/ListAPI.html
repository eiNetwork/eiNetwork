addTitlesToList (line 966)

Add titles to a user list.

Parameters:

  • username - The barcode of the user. Can be truncated to the last 7 or 9 digits.
  • password - The pin number for the user.
  • listId - The id of the list to add items to.
  • recordIds - The id of the record(s) to add to the list.
  • tags - A comma separated string of tags to apply to the titles within the list. (optional)
  • notes - descriptive text to apply to the titles. Can be viewed while on the list. (optional)

Note: You may also provide the parameters to addTitlesToList and titles will be added to the list after the list is created.

Returns:

  • success – true if the account is valid and the titles could be added to the list, false if the username or password were incorrect or the list could not be created.
  • listId – the id of the list that titles were added to.
  • numAdded – the number of titles that were added to the list.

Sample Call:

  1.  http://catalog.douglascountylibraries.org/API/ListAPI?method=createList&username=23025003575917&password=1234&title=Test+List&description=Test&public=0

Sample Response:

  1.  {"result":{"success":true,"listId":"1688"}}

createList (line 907)

Create a User list for the user.

Parameters:

  • username - The barcode of the user. Can be truncated to the last 7 or 9 digits.
  • password - The pin number for the user.
  • title - The title of the list to create.
  • description - A description for the list (optional).
  • public - Set to true or 1 if the list should be public. (optional, defaults to private).

Note: You may also provide the parameters to addTitlesToList and titles will be added to the list after the list is created.

Returns:

  • success – true if the account is valid and the list could be created, false if the username or password were incorrect or the list could not be created.
  • listId – te id of the new list that is created.

Sample Call:

  1.  http://catalog.douglascountylibraries.org/API/ListAPI?method=createList&username=23025003575917&password=1234&title=Test+List&description=Test&public=0

Sample Response:

  1.  {"result":{"success":true,"listId":"1688"}}
